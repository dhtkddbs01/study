# 파이차트

- autopct = 숫자를 같이 출력(1.1f -> 소수 첫째점 까지)
- shadow = 그림자 유무
- explode = 부채꼴이 파이 차트의 중심에서 벗어나는 정도
- labels = 데이터 라벨
- colors = 색상 지정

```python
# 소수점 첫째까지 값을 표시
# 그림자는 있도록
# 첫번째 부채꼴 그림은 0.1 정도 튀어나오게
plt.pie(df['EmployeeNumber'], labels=df['EmployeeNumber'].index, autopct='%1.1f%%',
         shadow= True, explode=[0.1, 0])
plt.title('업무 성과 분포')
plt.show()
```
