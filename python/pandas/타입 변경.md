# 타입 변환
## 1. 타입 확인
### 1.1. dtype()
- 열의 타입을 Series로 반환
```python
# 데이터프레임 열들의 타입을 반환
df.dtypes
```
### 1.2. select_dtypes()
- 특정 타입을 가진 컬럼만 추출
```python
# int 타입인 컬럼만 추출
df.select_dtypes('int')
# object 타입인 컬럼만 추출
df.select_dtypes('object')
```
