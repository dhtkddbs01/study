# 결측치 처리
NAME이 결측치(NULL)일 경우 'NO NAME'으로 변경해서 출력
## 1.COALESCE
```SQL
SELECT ANIMAL_TYPE, 
       COALESCE(NAME, 'No name') AS NAME, 
       SEX_UPON_INTAKE 
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```
 
## 2. IFNULL 
```SQL
SELECT ANIMAL_TYPE, 
       IFNULL(NAME, 'No name') AS NAME, 
       SEX_UPON_INTAKE 
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```
 
 
## 3. CASE표현식
```SQL
SELECT ANIMAL_TYPE, 
       CASE 
           WHEN NAME IS NULL THEN 'No name' 
           ELSE NAME 
       END AS NAME, 
       SEX_UPON_INTAKE 
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```
