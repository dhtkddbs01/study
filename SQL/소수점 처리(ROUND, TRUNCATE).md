# ROUND
- 숫자를 반올림
- ROUND(숫자, 반올림할 자릿수)

```SQL 
SELECT ROUND(3456.1234567) FROM DUAL
-- 3456 -- 

ELECT ROUND(3456.1234567 ,4) FROM DUAL
-- 345.1235 -- 

SELECT ROUND(3456.1234567 ,-2) FROM DUAL
-- 3500 --
```
# TRUNCATE
숫자를 버림
- TRUNCATE(숫자, 버릴 자릿수)

```SQL
SELECT TRUNCATE(3456.1234567 ,4) FROM DUAL;
-- 3456.1234 --

SELECT TRUNCATE(3456.1234567 ,-1) FROM DUAL;
-- 3450 -- 
```
